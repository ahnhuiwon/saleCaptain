    ## 🧐 프로젝트 빌드 및 실행 방법

    1. 상단 `Code` 버튼을 눌러 레포지토리를 클론 받습니다.

    ```
    $ git clone https://github.com/ahnhuiwon/saleCaptain.git
    ```

    2. 패키지를 설치합니다.

    ```
    $ npm install
    ```

    3. node JS를 실행합니다.

    ```
    $ node ./server/server.js
    ```
    
    3. React 를 실행합니다.

    ```
    $ npm start
    ```
